life-cycle-hooks Concept

events that are fired by Angular for components,directives,pipes etc...

1. life cycle hooks are interface that you need to implement it

2. life cycle hooks are fired by angular

Life cycle Hooks

all life cycle hooks methods are prefix with ng .see in below table

Angular calls cycle hooks methods on directives and components as it creates,change,and destroy them.

creates method changes/update method destroy method
ngOnInit : after the first ngOnChanges ngDocheck:developer's custom change detection ngOnDestroy: just before the directive is destroyed
ngAfterContentInit: After component content initialized ngOnChanges: called when input or output binding value changes
ngAfterViewInit: after component's view(s) are initialized ngAfterContentChecked: after every check on component content
ngAfterViewChecked:after every check of component's view(s)

Content vs Template

hello world is the content for p tag

like table tag with all inside tag are template

So if you want to access content of tag then you need to subscribe to ngAfterContentInit
and alongwith this you can used decorator such as @ContentChild / @ContentChildren

======================================================================================

So if you want to access Template of tag(s) then you need to subscribe to ngAfterViewInit
and alongwith this you can used decorator such as @ViewChild / @ViewChildren
