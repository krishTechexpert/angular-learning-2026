Switch case demo

💡 Notice [ngSwitch] uses [ ] (property binding) while *ngSwitchCase uses * (structural directive) — common gotcha for React devs.

Why Single Quotes Inside Double?

html*ngSwitchCase="'admin'"

The outer double quotes " " — tell Angular this is a template expression (like JS eval)

The inner single quotes ' ' — tell Angular this is a string literal inside that expression

What Happens Without Inner Quotes?

❌ Wrong — Angular treats admin as a VARIABLE

*ngSwitchCase="admin"

✅ Correct — Angular treats 'admin' as a STRING

*ngSwitchCase="'admin'"

👑 Admin — Full access

✏️ Editor — Can edit content

👁️ Viewer — Read only

🚫 Unknown role — No access
