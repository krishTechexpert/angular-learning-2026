Pipe Concept:

What is a Pipe?

Data Transformation --> Think of it like a filter in a Instagram photo — same photo, different look.
Same data, different display.

Original Data → Pipe → Formatted Output

'krishna' → uppercase → 'KRISHNA'

1000000 → currency → '₹10,00,000'

'2024-01-15' → date → 'Jan 15, 2024'

React vs Angular

In React — you format data manually:

React — manual formatting everywhere

name.toUpperCase()

₹price.toLocaleString('en-IN')

new Date(date).toLocaleDateString()

Same logic repeated in every component 😫

In Angular — one pipe, use anywhere: In built feature

name | uppercase

price | currency:'INR'

date | date:'mediumDate'

When Do You Need It?

Displaying formatted date → date pipe

Showing currency → currency pipe

Uppercase / lowercase text → uppercase / lowercase pipe

Filtering a list → custom pipe

Limiting text length → custom pipe

Converting JSON to string → json pipe

decimal pipe: minIntegerDigit.minFractionDigit-maxFractionDigit

Example: {{num1 | number}}

minIntegerDigit: Minimum number of interger digit. default is 1

minFractionDigit: Minimum number of fraction digit. default is 0

maxFractionDigit: Minimum number of fraction digit. default is 3

Example: {{num1 | number}}

Example: {{num1 | number:'3.2-5'}} //012.63847

before decimal, minimum number should be 3 so angular add 0 automatically 012.

another decimal example

{{x | number:'3.2-5'}} // 00.50
